My Home Server

My home server is a computer that I set up in mid October of 2024. It runs a few things, including:

  • Plex (Movie streaming)
  • Crafty (Minecraft Servers)
  • BeamMP (A multiplayer server for BeamNG.drive)
  • Samba (File sharing)

Specs (For Nerds)

My home server is an old Dell XPS 630i from 2006. I picked it up in 2021 from the tip shop in Naenae for $20.
It boasts an Intel Core 2 Quad Q8300 with 4 cores, clocked at 2.5ghz.
It runs Ubuntu 22.04LTS.
It has 120gb of SSD storage (boot and apps only).
I have 4 3TB HDDs set up in a RAID 5 array (redundant array of inexpensive disks) so that 1 of the HDDs can fail and I wouldn't lose any data.
